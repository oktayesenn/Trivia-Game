## Preview

This is a game of trivia basic style, with score counter.

## Technology used:

- I used vanilla javascript, css and html.
- I used some locally stored images and fonts.


## Approach:
The main approach to the game is using questions with arrays.
And for point system: Each correct question is 1 point.
### This is self challenge:
i saved the entire state of the game.

## Main features:
- Icon changer.
- Audio for the tics.
- Background color
- Timer


## User Stories
- As a user, I should be able to click and see the correct answer or incorrect.
- As a user, I should be shown a message after each turn for if I win, lose.
- As a user, I should be shown a message when I win, lose or tie
- As a user, I should not be able to continue playing once I win, lose, or tie
- As a user, I should be able to play the game again without refreshing the page
